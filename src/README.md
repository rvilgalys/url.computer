# url.computer

A luxury developer tool for parsing URLs and building cURL commands. Runs entirely in the browser for privacy and instant performance.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to use the tool.

## Scripts

- `npm run dev` — Development server with Turbopack
- `npm run build` — Production build
- `npm start` — Production server
- `npm run lint` — ESLint
- `npm test` — Jest test suite

## Deployment

Deployed on [Vercel](https://vercel.com). Push to `main` to deploy.
